# dk_sdfi_README.txt

The files in this section result from the conversion of datasets originating
from [Agency for Data Supply and Infrastructure](https://sdfi.dk) (previously known as SDFE).

## Included grids

### ETRS89 -> GS-IRF

*Source*: [Agency for Data Supply and Infrastructure](https://sdfi.dki/vores-opgaver/geodaesi/koordinattransformation)
*Format*: GTG
*License*: [Creative Commons Attribution 4.0](https://creativecommons.org/licenses/by/4.0/)

Transform ETRS89 coordinates (EPSG:4258) to Generalstabens Systems intermediate reference
frame (EPSG:1346), which is used as a stepping stone in the transformation between ETRS89
and GS.

* dk_sdfi_gs_2022.tif
